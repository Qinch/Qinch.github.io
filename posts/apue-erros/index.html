APUE中文第三版勘误

  • Oct 2, 2017

本博客根据自己对APUE中文第三版的阅读理解，列出其中自己认为翻译错误或者翻译有歧义的部分。

1,page81

中文：其中，如果测试文件是否已经存在，mode就为F_OK；否则mode是图4-7中所列常量的按位或。 英文(page102)：The mode is either the value F_OK to test if a file exists, or the bitwise OR of any of the flags shown in Figure 4.7.

2,page89

中文：对于符号链接，文件长度是在文件名中的实际字节数。 英文(page111):For a symbolic link, the file size is the numberof bytes in the filename.
